# 組版アプリ v017

GitHub / Vercelで公開できる、ローカル完結型の出版社向け組版アプリです。

## v017の主な追加

- PDF保存前の「出力前チェック」
- エラー・要確認・情報の3段階表示
- 用紙サイズ、余白、文字サイズ、行間の確認
- タイトル・本文・目次・見出し階層の確認
- まえがき、あとがき、著者紹介、奥付の未入力確認
- 画像参照切れ、未使用画像、推定解像度の確認
- 空白ページ、表示できない画像、本文領域のはみ出し検出
- 各チェック結果から修正箇所へ移動
- エラーがある場合はPDF保存前にチェック画面を表示
- v016の保存プロジェクト・テンプレートを自動移行

## PDFについて

PDFはアプリ内のページを画像として固定する確認・共有向け形式です。
PCの印刷設定には依存しませんが、文字検索・コピー、PDF/X、CMYK、フォント埋め込み検査には対応していません。

## 公開方法

1. ZIPを展開します。
2. 展開したファイルをGitHubリポジトリへアップロードします。
3. VercelでリポジトリをImportします。
4. Framework Presetは「Other」のまま公開できます。

## 保存上の注意

ブラウザ保存は同じ端末・ブラウザ・公開URLでのみ利用できます。重要な案件はJSONも保存してください。
