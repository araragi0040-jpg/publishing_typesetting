# 組版アプリ v001

出版社向け組版アプリの技術検証版です。Next.jsやビルド環境を使わず、HTML・CSS・JavaScriptだけで動作します。

## v001の対象範囲

- 横書き
- 一段組
- タイトル、サブタイトル、著者名、本文
- A4、A5、B6、自由サイズ
- 上下左右余白
- フォント
- 文字サイズ
- 文字間
- 行間
- 段落字下げ
- 文字揃え
- 自動改ページ
- ページ番号
- 1ページ／見開きプレビュー
- ブラウザ内保存
- JSON書き出し／読み込み
- ブラウザ印刷によるPDF保存

## ローカルでの確認

`index.html` をChromeまたはEdgeで開いてください。

ファイルを直接開いた状態でも動作します。より本番に近い形で確認する場合は、簡易Webサーバーを使用してください。

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。

## GitHub・Vercelでの公開

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内のファイルをリポジトリへアップロードします。
3. Vercelで「Add New Project」を選択します。
4. 対象のGitHubリポジトリを選択します。
5. Framework Presetは「Other」を選択します。
6. Build Commandは空欄、Output Directoryも空欄のままデプロイします。

GitHubへ更新を反映すると、Vercel側も自動更新されます。

## PDF出力

アプリ右上の「PDF出力」を押します。

Chrome・Edgeの印刷画面で、以下を推奨します。

- 送信先：PDFに保存
- レイアウト：自動
- 余白：なし
- 倍率：100%
- 背景のグラフィック：オン
- ヘッダーとフッター：オフ

ブラウザやプリンタードライバーによっては、独自用紙サイズの扱いが異なる場合があります。

## 保存について

「端末に保存」は、ブラウザのlocalStorageを使用します。同じブラウザ・同じURLでのみ読み込めます。

別端末への移動やバックアップには「JSON出力」を使用してください。

## フォントについて

v001では、端末にインストールされているフォントを参照します。フォントファイルは同梱していません。

候補に表示されても端末に存在しない場合は、同系統の代替フォントが使用されます。実運用では、出版社が利用するフォントのWeb利用・PDF埋め込み・商用利用条件を確認する必要があります。
