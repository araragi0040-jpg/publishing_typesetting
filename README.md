# 組版アプリ v010

GitHub・Vercelで公開できる、GAS不要のローカル完結型組版アプリです。

## v010の主な追加機能

- 大見出し・中見出し・小見出しから目次を自動生成
- 目次へ載せる見出し階層を選択
- 目次タイトル、文字サイズ、行間を設定
- ページ番号と点線リーダーの表示切替
- 見出し名・順序・改ページ位置の変更時に目次を自動更新
- 目次を含め、アプリ内のレイアウトをそのままPDFへ直接保存
- v009の保存済みプロジェクトとテンプレートを自動移行

## 使い方

1. `index.html` を開くか、GitHubへアップロードしてVercelで公開します。
2. 全文編集へ原稿を貼り付けます。
3. `# 大見出し`、`## 中見出し`、`### 小見出し`を設定します。
4. 右側の「目次」を開き、「目次を表示する」をオンにします。
5. 目次へ載せる階層と表示設定を選びます。
6. 右上の「PDF保存」で、目次を含むPDFを直接保存します。

目次を表示する場合は、タイトル、目次、本文の順に独立して配置されます。目次内のページ番号は、ページ番号設定の開始番号を基準に自動計算されます。

## 保存について

プロジェクトとテンプレートはブラウザの `localStorage` へ保存されます。重要な案件はJSON出力も行ってください。別端末や別ブラウザへ移す場合はJSONファイルを使用します。

## ファイル構成

- `index.html`：画面構成
- `styles.css`：UI・誌面・目次のスタイル
- `app.js`：組版、章管理、目次、保存、PDF生成
- `vendor/`：PDF生成用ライブラリ
- `vercel.json`：Vercel静的配信設定

## PDFについて

v010はプレビューを画像として固定した確認・校正用PDFを生成します。PCの印刷設定には依存しません。PDF内の文字検索、PDF/X、CMYK、商業印刷用フォント検査には未対応です。
