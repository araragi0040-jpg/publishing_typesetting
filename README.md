# 組版アプリ v011

GitHub・Vercelで公開できる、GAS不要のローカル完結型組版アプリです。

## v011の主な追加機能

- 日本語組版設定を追加
- 行頭・行末禁則を「厳密／標準」から選択
- 句読点のぶら下がり表示を切替
- 段落がページをまたぐ際、1行だけがページ上部・下部へ残る状態を抑制
- ページ上下へ残す最低行数を1～3行で設定
- 見出し後に確保する本文行数を1～3行で設定
- 「おすすめ設定に戻す」ボタンを追加
- v010の保存済みプロジェクトとテンプレートを自動移行

## おすすめの使い方

日本語組版は初期状態のままで使用できます。

- 禁則処理：厳密
- 句読点のぶら下がり：有効
- 段落の1行残り防止：有効
- ページ上下に残す最低行数：2行
- 見出し後に確保する本文：2行

設定を変更して分かりにくくなった場合は、右側の「日本語組版」内にある「おすすめ設定に戻す」を押してください。

## 基本操作

1. `index.html` を開くか、GitHubへアップロードしてVercelで公開します。
2. 全文編集へ原稿を貼り付けるか、章別編集で章ごとに入力します。
3. 右側の組版設定を調整します。
4. 中央プレビューで改ページや見出し位置を確認します。
5. 右上の「PDF保存」で、アプリ内の表示をそのままPDFへ直接保存します。

## 保存について

プロジェクトとテンプレートはブラウザの `localStorage` へ保存されます。重要な案件はJSON出力も行ってください。別端末や別ブラウザへ移す場合はJSONファイルを使用します。

## ファイル構成

- `index.html`：画面構成
- `styles.css`：UI・誌面・日本語組版設定のスタイル
- `app.js`：組版、章管理、目次、日本語禁則、保存、PDF生成
- `vendor/`：PDF生成用ライブラリ
- `vercel.json`：Vercel静的配信設定

## PDFについて

v011はプレビューを画像として固定した確認・校正用PDFを生成します。PCの印刷設定には依存しません。PDF内の文字検索、PDF/X、CMYK、商業印刷用フォント検査には未対応です。
