# 組版アプリ v018

GitHub / Vercelで公開できる、ローカル完結型の出版社向け組版アプリです。

## v018の主な変更

- 初回起動時に「新規作成／JSON読込／サンプル」の開始画面を表示
- 横書き書籍・縦書き書籍・冊子から選べる新規作成ウィザード
- 画面上部に4段階の作業ナビを追加
- 主要操作を「作成・内容・確認・PDF」に整理
- 低頻度の操作を「その他」メニューへ整理
- 自動保存待ち・保存中・保存済み・失敗を上部へ明示
- 4ステップの簡易使い方ガイド
- 小さいPC画面でのツールバーと3カラム幅を調整
- v017の保存データとテンプレートを自動移行

## 基本操作

1. 「新しく作る」から書籍形式を選択
2. 原稿を全文貼り付け、または章別に入力
3. 中央プレビューを見ながら右側の組版設定を調整
4. 必要に応じて書籍構成・画像・目次を設定
5. 出力前チェック後にPDF保存

## 保存について

- 編集内容は同じブラウザへ自動保存されます。
- 重要な案件は「その他 → JSON出力」でバックアップしてください。
- 別端末ではJSONを読み込んでください。

## Vercel公開

このフォルダ一式をGitHubリポジトリへ配置し、Vercelで静的サイトとして公開できます。ビルドコマンドは不要です。
