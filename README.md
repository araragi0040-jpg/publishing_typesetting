# 組版アプリ v014

GitHub / Vercel でそのまま公開できる、ブラウザ完結型の簡易組版アプリです。GASや外部データベースは使用しません。

## v014の主な追加

- 選択した文字範囲への太字
- 選択した文字範囲への傍点
- 選択した文字範囲へのルビ
- 選択した文字範囲への下線
- 選択範囲の装飾解除
- 横書き・縦書き両方のプレビューとPDFへ反映
- 文字装飾の記号を除外した文字数表示
- v013のプロジェクト・テンプレート自動移行

## 基本操作

1. 全文編集または章別編集で、装飾したい文字を選択します。
2. 入力欄上部の「太字」「傍点」「ルビ」「下線」を押します。
3. ルビの場合は読みを入力します。
4. 中央プレビューで結果を確認します。
5. 「PDF保存」で、現在の組版結果を直接PDFへ保存します。

## 使用する記法

- 太字：`**文字**`
- 傍点：`《《文字》》`
- ルビ：`｜漢字《かんじ》`
- 下線：`__文字__`

通常は記法を手入力せず、文字を選択してボタンを使用してください。記法は原稿編集欄では確認できますが、プレビューとPDFには表示されません。

## 保存について

- 編集中データ：同じブラウザの localStorage
- 受け渡し・バックアップ：JSON出力
- 完成確認：PDF保存

重要な案件は、作業の区切りごとにJSONを保存してください。
