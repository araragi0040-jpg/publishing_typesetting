# 組版アプリ v012

GitHub と Vercel で公開できる、ブラウザ完結型の簡易組版アプリです。GASや外部データベースは使用しません。

## v012の主な追加内容

- 書籍情報と本文を対象にした原稿チェック
- タブ、特殊空白、見えない文字、行末空白の検出
- 手入力の字下げ、連続スペース、過剰な空行の検出
- 三点リーダー、ダッシュ、句読点、全角英数字の確認
- 見出し記号の入力ミス、空見出しの検出
- 括弧の閉じ忘れ・開き忘れの検出
- 結果から全文編集の該当箇所へ移動
- 項目ごとの修正と、安全な項目だけの一括修正
- v011の保存プロジェクトとテンプレートを自動移行

原稿チェックは勝手に文章を書き換えません。修正ボタンを押した項目だけ変更します。

## 使い方

1. `index.html` をブラウザで開くか、GitHubへアップロードしてVercelで公開します。
2. タイトル、著者名、本文を入力します。
3. 右上の「原稿チェック」を押し、確認項目を確認します。
4. 必要な項目だけ修正し、組版設定を調整します。
5. 「PDF保存」でアプリ内プレビューを直接PDF化します。
6. 作業区切りごとにJSONも出力してバックアップします。

## ファイル構成

- `index.html`：画面構成
- `styles.css`：UI・誌面・PDF描画用スタイル
- `app.js`：組版、保存、章管理、目次、日本語組版、原稿チェック、PDF生成
- `vendor/`：PDF生成用ライブラリ
- `vercel.json`：Vercel静的公開設定

## 保存について

プロジェクトとテンプレートは、同じブラウザ・同じ公開URLの `localStorage` に保存されます。ブラウザデータの削除や端末故障に備え、重要な案件はJSONを別途保存してください。
