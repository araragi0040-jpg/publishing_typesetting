# 組版アプリ v013

GitHub と Vercel で公開できる、ブラウザ完結型の簡易組版アプリです。GASや外部データベースは使用しません。

## v013の主な追加内容

- 横書き・左綴じ／縦書き・右綴じの切替
- 縦書き時の右から左への見開き表示
- 綴じ方向に合わせた内側・外側余白の自動入替
- 縦書き時の英数字方向の選択
- 1〜3桁の半角数字の縦中横
- 縦書きの本文、見出し、タイトル、目次、ページ番号、ヘッダー・フッター
- 縦書きレイアウトをそのまま直接PDF保存
- 「A5 縦書き・右綴じ」標準テンプレート
- v012の保存プロジェクトとテンプレートの自動移行

## 使い方

1. `index.html` をブラウザで開くか、GitHubへアップロードしてVercelで公開します。
2. 右側の「用紙・文字方向」で、横書きまたは縦書きを選びます。
3. タイトル、著者名、本文を入力し、必要に応じて目次や見出しを設定します。
4. 中央プレビューで改ページと見開きを確認します。
5. 「PDF保存」でアプリ内の縦書き・横書きレイアウトを直接PDF化します。
6. 作業区切りごとにJSONも出力してバックアップします。

## 余白について

左右の入力欄は「内側・綴じ側」と「外側」です。横書き・左綴じ、縦書き・右綴じのどちらでも、奇数・偶数ページに合わせて物理的な左右余白を自動で入れ替えます。

## 保存について

プロジェクトとテンプレートは、同じブラウザ・同じ公開URLの `localStorage` に保存されます。重要な案件はJSONも保存してください。
