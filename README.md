# 組版アプリ v020 修正版

GitHub / Vercelで公開できる、出版社向けのローカル完結型組版アプリです。

## v020の追加内容

- 本文画像とは別の「装飾画像」を追加
- 現在表示中のページへ自由配置
- プレビュー上でドラッグ移動
- 右下ハンドルで拡大・縮小
- X・Y位置と幅をmm単位で数値調整
- 本文の前面／背面を切替
- 不透明度を調整
- 配置ロック、複製、削除
- ページ移動・中央揃え
- ブラウザ保存、JSON、戻す／やり直すへ対応
- 装飾画像をPDFへそのまま反映

## 基本操作

1. 上部の「画像・装飾」を開きます。
2. 画像をアップロードします。
3. 画像カードの「装飾として配置」を押します。
4. 中央プレビューでドラッグ移動し、右下の丸いハンドルで拡大・縮小します。
5. 選択バーの「数値で調整」から、ページ、位置、幅、不透明度、前面／背面、ロックを設定できます。

本文の流れに沿わせる図版は、従来どおり「本文へ挿入」を使用してください。

## 公開

このフォルダ一式をGitHubへアップロードし、Vercelで静的サイトとして公開できます。

## 保存

原稿・画像・装飾配置はブラウザ内に保存されます。重要な案件はJSON出力も行ってください。

## 修正内容

- 縦書きPDFで英字が反転・位置ずれする問題を修正
- ヘッダーを横書き固定し、外側・内側・中央の配置をPDFでも維持
- 縦書きの三点リーダー「……」の中心位置を調整
- 半角2〜3桁の縦中横を横並びで表示
- 字下げを段落先頭だけでなく、本文中の改行先頭にも適用
- 「」などの記号から始まる行は自動的に字下げしない
- 文中の「！」「？」の後へ全角スペースを自動挿入（文末・閉じ括弧前を除く）
- 文字装飾欄へ「……」挿入ボタンを追加
