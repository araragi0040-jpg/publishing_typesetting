# 組版アプリ v019.1

GitHub / Vercelで公開できる、ローカル完結型の出版社向け組版アプリです。

## v019の主な変更

- 原稿・組版・書籍構成を最大40段階まで元に戻す／やり直す
- 本文のみ、または書籍全体を対象にした検索・置換
- 検索結果から該当箇所へ直接移動
- プレビュー上の前後ページ移動とページ番号指定
- 大・中・小見出し一覧から誌面・原稿へ移動
- 指定した連続ページだけをPDF保存
- v018の保存データとテンプレートを自動移行

## 基本操作

1. 「新しく作る」から書籍形式を選択
2. 原稿を全文貼り付け、または章別に入力
3. 中央プレビューを見ながら右側の組版設定を調整
4. 「検索・移動」から検索、見出し移動、範囲PDFを使用
5. 出力前チェック後にPDF保存

## 編集履歴について

- 「戻す」「やり直す」は、原稿・組版・前付後付・画像設定などの編集状態を保持します。
- 最大40段階です。
- 画像本体は同一セッション内で保持し、画像設定や削除も復元できるようにしています。
- プロジェクトを開き直した時点で履歴はリセットされます。

## 保存について

- 編集内容は同じブラウザへ自動保存されます。
- 重要な案件は「その他 → JSON出力」でバックアップしてください。
- 別端末ではJSONを読み込んでください。

## Vercel公開

このフォルダ一式をGitHubリポジトリへ配置し、Vercelで静的サイトとして公開できます。ビルドコマンドは不要です。


## v019.1 縦書きPDF修正

縦書きPDFで読点・句点・括弧・英数字・縦中横の位置が崩れる問題を修正しました。

- 横書きPDF：従来の描画方式を維持
- 縦書きPDF：ブラウザ本来の縦書き描画を使用
- プレビュー、ページ分割、余白、目次、ルビ、傍点、画像設定は変更なし
- 保存キーとデータ形式はv019のままなので、既存プロジェクトをそのまま引き継げます

ChromeまたはEdgeの最新版での利用を推奨します。
