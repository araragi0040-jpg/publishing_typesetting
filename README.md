# 組版アプリ v015

GitHub / Vercelで公開できる、ブラウザ完結型の簡易組版アプリです。GASや外部データベースは使用しません。

## v015の主な追加

- JPEG・PNG・WebP画像のアップロード
- アップロード画像の自動最適化
- 全文編集・章別編集のカーソル位置への画像挿入
- 表示幅（25～100%）と左右・中央配置
- キャプション・代替テキスト
- 画像前後の余白と画像前改ページ
- 横書き・縦書きプレビューへの反映
- アプリ内直接PDFへの画像・キャプション反映
- プレビュー画像をクリックして設定画面を開く操作
- 画像データをブラウザ保存・JSONへ包含
- v014のプロジェクト・テンプレート自動移行

## 基本操作

1. 全文編集または章別編集で、画像を置きたい位置へカーソルを移動します。
2. 文字装飾欄の「画像」を押します。
3. 「画像をアップロード」からファイルを選びます。
4. 必要に応じて幅、配置、キャプションを設定します。
5. 「本文へ挿入」を押します。
6. 中央プレビューで確認し、「PDF保存」で出力します。

本文編集欄には `[[figure:...]]` という位置記号が入ります。この記号はプレビューとPDFには表示されません。削除・移動するときは、通常の文章と同様に行ごと操作できます。

## 画像保存の注意

画像はブラウザ保存とJSONへ含めるため、アップロード時に最大辺2000pxを目安に自動最適化します。1プロジェクト20点、最適化後データ約3.2MBを上限としています。

v015のPDFは確認・校正用の画像固定型PDFです。商業印刷向けの原寸高解像度画像管理やCMYK、PDF/Xは今後の入稿対応工程で扱います。

## 保存について

- 編集中データ：同じブラウザのlocalStorage
- 受け渡し・バックアップ：JSON出力
- 完成確認：PDF保存

画像入り案件はデータ量が大きくなるため、作業の区切りごとにJSONを保存してください。
